# Dementia Prev
Fight dementia with us!

## Getting Started
1. Clone the repository 
    ```
    git clone https://github.com/ekoeditaa/dementia-prev.git
    ```
2. Install dependencies
    ```
    yarn install
    ```
3. Run the project
    ```
    yarn start
    ```
